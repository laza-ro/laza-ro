### Importador de Planilhas com Cálculo de CSAT

Funcionalidade para importar planilhas, validar dados por mês selecionado e calcular percentual de satisfação dos atendimentos.

**Principais recursos:**
- Validação de período mensal
- Cálculo de CSAT com base nas notas 4 e 5
- Associação dos dados importados com usuários cadastrados
- Processamento de dados vindos de planilhas Excel

**Tecnologias utilizadas:** JavaScript, Node.js, Excel, SQL
